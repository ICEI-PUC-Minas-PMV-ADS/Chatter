# Instruções de utilização

 Para inicialização do app deve ser feito so seguintes passos:

cd server 
- npm install
- npm i express mongoose nodemon socket.io bcrypt cors dotenv
- npm start

cd public
- npm install
- npm i express mongoose nodemon socket.io bcrypt cors dotenv
- npm start

 Agora seu projeto ja esta rodando!
